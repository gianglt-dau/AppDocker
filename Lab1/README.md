# Lab 1: Xây dựng Backend NodeJS & Frontend ReactJS (Mock Data)

## Mục tiêu
- Tạo API NodeJS trả về dữ liệu giả (mock data)
- Tạo Frontend ReactJS gọi API và hiển thị dữ liệu

## Hướng dẫn chạy

### 1. Chạy NodeJS API
```bash
cd node-api
npm install
node app.js
```
API sẽ chạy tại: http://localhost:3000

### 2. Chạy Frontend React
```bash
cd frontend
npm install
npm run dev
```
Frontend sẽ chạy tại: http://localhost:5173

### 3. Kiểm tra
- Mở trình duyệt tại http://localhost:5173
- Bạn sẽ thấy danh sách task từ NodeJS API
